# Khai Serapio — Multimedia Creative Portfolio

Award-winning multimedia professional with 3+ years of expertise in video production, motion design, and visual storytelling.

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lenis (smooth scrolling)
- HLS.js (video streaming)
